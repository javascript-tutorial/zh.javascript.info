**答案：在这两种情况下都是从 `0` 到 `4`。**

```js run
for (let i = 0; i < 5; ++i) alert( i );

for (let i = 0; i < 5; i++) alert( i );
```

这可以很容易地从 `for` 算法中推导出：

1. 在一切开始之前执行 `i = 0`。
2. 检查 `i < 5` 条件
3. 如果 `true` —— 执行循环体并 `alert(i)`，然后进行 `i++`

递增 `i++` 与检查条件（2）分开。这只是另一种写法。

在这没使用返回的递增值，因此 `i++` 和 `++i`之间没有区别。
