# 使用 '?' 重写 'if..else' 语句

使用多个三元运算符 `'?'` 重写下面的 `if..else` 语句。

为了增强代码可读性，建议将代码分成多行。

```js
let message;

if (login == 'Employee') {
  message = 'Hello';
} else if (login == 'Director') {
  message = 'Greetings';
} else if (login == '') {
  message = 'No login';
} else {
  message = '';
}
```
