# Changing "prototype"

在下面的代码中，我们创建了 `new Rabbit`，然后尝试修改其原型。

一开始，我们有这样的代码：

```js run
function Rabbit() {}
Rabbit.prototype = {
  eats: true
};

let rabbit = new Rabbit();

alert( rabbit.eats ); // true
```
