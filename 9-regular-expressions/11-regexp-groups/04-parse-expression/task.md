# 解析表达式

一个算术表达式包含两个数字和一个中间的操作符，比如：

- `1 + 2`
- `1.2 * 3.4`
- `-3 / -6`
- `-2 - 2`


操作符是 `"+"`, `"-"`, `"*"`， `"/"` 其中之一。

在开头，中间或结尾可能包含额外的空格。

创建一个函数 `parse(expr)`，以一个算术表达式作为参数，返回包含3项的数组：

1. 第一个数字
2. 操作符
3. 第二个数字

用例：

```js
let [a, op, b] = parse("1.2 * 3.4");

alert(a); // 1.2
alert(op); // *
alert(b); // 3.4
```
