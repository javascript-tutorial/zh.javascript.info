# Up/down 按钮

创建一个“到顶部”按钮来帮助页面滚动。

它应该像这样运行：
- 页面向下滚动的距离没有超过窗口高度时 —— 按钮不可见。
- 当页面向下滚动距离超过窗口高度时 —— 在左上角出现一个“向上”的箭头。如果页面回滚回去，箭头就会消失。
- 单击箭头时，页面将滚动到顶部。
